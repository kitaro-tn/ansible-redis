# ansible-redis

## Description
It's a redis that was constructed in the ansible

## Environment

### Execution environment
- vagrant (>= 1.7)
- serverspec (>= 2.17)
- ansible (>= 2.0.0)

### Build OS
- CentOS 7

## Vagrant setup

```bash
$ vagrant up
```

## Testing

```bash
$ rake spec:_default
```
